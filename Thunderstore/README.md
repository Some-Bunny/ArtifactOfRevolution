# Artifact Of Revolution

A basic Artifact functionally similar to Evolution, but with a few twists and turns. 
-

![Revolution Artifact Showcase](https://imgur.com/CSuIqx1.gif)
- Gives enemies items. Enemy item loadout changes every stage, and has a few twists and turns that may benefit or end your run.

- Has a LOT of config options. Some options / features will only function with DLCs *(Mainly, Survivors Of The Void & Alloyed Collective)*, but the mods base functionality should work without any DLCs.

Commissioned by a friend!
